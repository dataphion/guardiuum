## Data Sources

Guardiuum supports various different standard and custom data as source. Supported connectors for the current version are:

- Google BigQuery
- Databricks
- Object Storage such as S3 and ABFS (Datalake)
- Hive
- MariaDB
- MongoDB
- MySQL
- Postgres
- Presto
- Starburst
- Trino
- Redshift
- Snowflake
- Salesforce (Enterprise Products)
